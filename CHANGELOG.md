## [1.0.0] - Initial Portfolio Launch

### Added
- Full photography portfolio website
- Image carousel with 25+ images
- GSAP scroll animations
- Lenis smooth scrolling
- Dark theme with gold accents
- Responsive design
- Accessibility features (ARIA, keyboard nav, reduced motion)

### Tech Stack
- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- GSAP + ScrollTrigger
- TypeScript
- pnpm
