# Project Context — Forward Design Portfolio

**Project Name:** Forward Design — Rodrigue Ilunga Portfolio
**Project ID:** WF-2026-04-11-001
**Created:** 2026-04-11
**Status:** Active — Build Phase

## Goal
Build a premium, Awwwards-quality cinematic portfolio website for Rodrigue Ilunga (Forward Design) — a high-end photographer and visual artist from Kinshasa, DRC.

## Tech Stack
- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS + CSS Modules
- **Animations:** GSAP + Lenis Smooth Scroll + Framer Motion
- **Language:** TypeScript
- **Deployment:** Vercel (target)

## Design System
- **Background:** `#0a0a0a` to `#1a1a1a` (deep charcoal)
- **Primary Text:** `#f5f0eb` (warm white)
- **Accent:** `#d4a843` (amber/gold)
- **Secondary Accent:** `#c8956c` (warm skin tone)
- **CTA:** `#c67a3c` (burnt orange/copper)
- **Metadata:** `#6b6b6b` (warm gray)
- **Headings:** Playfair Display (serif)
- **Body:** Inter / Satoshi (sans-serif)

## Site Structure
1. Home/Hero — Cinematic 24-frame sequence + tagline
2. Portfolio/Work — Filterable asymmetric gallery
3. About — Bio, philosophy, Kinshasa
4. Social Impact — Story-driven project presentations
5. Contact — Warm CTA "Tell me your story"
